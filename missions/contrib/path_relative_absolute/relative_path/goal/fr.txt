Objectif
========

Dans la cours du château Lady Squirrel voudrait aller au Jardin des fleurs sans avoir à s'arrêter en chemin.

Pour réussir cette mission vous devez déplacer Lady Lux au Jardin des fleurs en une seule commande en prenant le chemin à partir du château où vous resterez..

Commandes utiles
================

mv FICHIER1 FICHIER2 ... FICHIERn REPERTOIRE
  Déplace les fichiers vers le répertoire.
  Note : ``mv`` est une abréviation pour "move".

..
  Le symbole ".." represente le répertoire parent (le lieu qui se trouve
  "avant" le lieu actuel dans la liste donnée par ``pwd``). Il est souvent utilié pour les chemins reltifs (chemin à partir de où je me trouve.)

ls
  Liste les lieux accessibles depuis la position actuelle.
  Note : ``ls`` est une abréviation pour "liste".

gsh check
  Vérifie si l'objectif de la mission a été atteint.


Remarque
--------

Les termes apparaissant en MAJUSCULES dans les commandes sont des 
méta-variables : vous devez les remplacer par des valeurs (chaines 
de caractères) appropriées.

La plupart des systèmes de fichiers différencient les caractères 
majuscules et minuscules. Faites attention à utiliser le bon chemin.
