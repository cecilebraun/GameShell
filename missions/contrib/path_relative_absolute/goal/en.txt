Mission goal
============

Lady Squirrel is waiting for you in the castle. Go and meet her. She want to go to the flowers Garden without taking any break.

To succeed you need to move Lady Lux in only one command with a direct path from the castle while you stay in the castle.


Useful commands
===============

mv FILE1 FILE2 ... FILEn DIRECTORY
  Move the files to the directory.
  Remark: ``mv`` is an abbreviation of "move".
..
  The ".." symbol represent the parent directory (one step back along the path to your current location). It is often used for relaive path (path described from where I stand.)

ls
  Show a list of locations that are currently accessible.
  Remark: ``ls`` is an abbreviation of "list".

gsh check
  Check if the mission objective has been achieved.


Remarks
-------

UPPERCASE words appearing in commands are meta-variables: you need
to replace them by appropriate (string) values.

Most filesystems treat uppercase and lowercase characters
differently. Make sure you use the correct path.
